Getting Started with Convex Optimization in Python

A practical guide to CVXPY and disciplined convex programming

Convex optimization is a powerful tool for solving a wide range of problems in engineering, finance, and machine learning. This tutorial introduces CVXPY, a Python library that makes convex optimization accessible and intuitive.

Why Convex Optimization?

Unlike general nonlinear optimization, convex problems have a beautiful property: any local minimum is also a global minimum. This means we can solve them efficiently and reliably, even for large-scale problems with thousands of variables.

Many practical problems can be formulated as convex programs, including portfolio optimization, signal processing, control systems design, and regularized regression.

Setting Up CVXPY

Installation is straightforward with pip. CVXPY provides a natural syntax for specifying optimization problems that closely mirrors mathematical notation.

The library automatically checks whether your problem is convex using disciplined convex programming (DCP) rules, alerting you if you accidentally specify a non-convex problem.

Your First Optimization Problem

Let's start with a simple least squares problem. We want to find a vector x that minimizes the squared Euclidean norm of Ax - b, where A is a matrix and b is a vector.

CVXPY allows us to express this naturally: we define a variable, construct an objective function, and call solve(). The library handles the rest, selecting an appropriate solver and returning the optimal solution.

Adding Constraints

Real-world problems often have constraints. CVXPY supports a rich set of constraint types:

  • Linear equality and inequality constraints
  • Second-order cone constraints
  • Semidefinite constraints
  • Exponential cone constraints

Advanced Topics

Once comfortable with basics, explore parametric optimization, warm starting for faster solves, and custom solver options. CVXPY also integrates well with NumPy and can handle sparse matrices for large-scale problems.

Next Steps

I recommend working through the examples on the CVXPY website and reading Boyd and Vandenberghe's textbook for theoretical foundations. Practice by formulating problems from your own domain as convex programs.
